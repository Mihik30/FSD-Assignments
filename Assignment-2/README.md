# Assignment 2 - Personal Portfolio Page

## Description

This assignment is a personal portfolio webpage designed using HTML and CSS in
a terminal-inspired theme. It presents academic background, internship
experience, projects, skills, achievements, and contact details in a developer
console style.

## Portfolio Highlights

- terminal-style interface with window chrome and prompt-based headings
- summary, education, experience, projects, skills, and achievements sections
- GitHub-backed project cards using public repository information from
  `github.com/Mihik30`
- responsive layout for desktop and mobile screens

## Technologies Used

- HTML5
- CSS3

## Files

- `index.html` - portfolio structure and content
- `styles.css` - terminal theme styling

## Notes

- GitHub was used as the primary source for public project links and repository
  names where available.
- Resume content was used for experience, education, skills, and achievements.

## How To Run

Open `index.html` in any browser.
