# Assignment 4 - Interactive Weather Dashboard

## Description

This assignment is an interactive weather dashboard built with HTML, CSS,
JavaScript, and Chart.js. It visualizes weekly weather patterns in a clean
dashboard format and supports simulated real-time updates with one click.

## Features

- interactive weather dashboard with responsive layout
- line chart for weekly temperature trends
- bar chart for weekly humidity levels
- pie chart for rainfall distribution across the week
- dynamic updates using JavaScript data simulation
- refresh button to simulate weather feed changes
- city rotation with uncommon Indian city datasets:
  Shillong, Leh, Kochi, Jaisalmer, Gangtok
- quick KPI cards for average temperature, average humidity, and total rainfall

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

## Files

- `index.html` - dashboard structure and chart containers
- `style.css` - visual styling, responsive layout, and animation
- `script.js` - Chart.js setup and dynamic data refresh logic

## How To Run

Open `index.html` in any browser.
