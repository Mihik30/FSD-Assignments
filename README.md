# FSD Assignments

Static front-end assignments built with HTML5 and CSS3.

## Assignments Included

### Assignment 1 - Student Table and HTML Tags Table
- Entry file: `Assignment-1/index.html`
- Focus: basic HTML structure, table markup, and CSS styling
- Features: student data table, HTML tags reference table, responsive layout, hover effects

### Assignment 2 - Personal Portfolio Page
- Entry file: `Assignment-2/index.html`
- Focus: personal portfolio design with HTML and CSS
- Theme: terminal-inspired interface with command/output styling
- Features: about section, education, experience, projects, skills, achievements, and contact links

### Assignment 3 - AI Model Storefront
- Entry file: `Assignment-3/index.html`
- Focus: frontend-only ecommerce design for AI products
- Theme: futuristic clean storefront with local-vs-hosted model comparisons
- Features: hero banner, product catalog, comparison table, pricing section, testimonials, and FAQ
