# Assignment 3 - AI Model Storefront

## Description

This assignment is a frontend-only AI ecommerce website designed like a real
online marketplace. The page uses a shopping-style header, search bar,
department cards, product shelves, pricing offers, and cart-style calls to
action while keeping the products centered around AI models and tools.

## Features

- ecommerce-style header with search and cart area
- department section and dense product listing grid
- shopping-style product cards with prices and CTA buttons
- comparison section for local models vs hosted APIs
- pricing offers and FAQ section
- responsive design built entirely with HTML and CSS

## Concept

The storefront presents AI models as products that can be browsed like items in
an online store and evaluated by:

- privacy and deployment control
- setup effort
- scalability
- cost model
- use-case fit

## Technologies Used

- HTML5
- CSS3

## Files

- `index.html` - storefront structure and content
- `styles.css` - visual design, layout, and responsive styling

## How To Run

Open `index.html` in any browser.
